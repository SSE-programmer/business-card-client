# Soft Mall Calendar — Employee Calendar

A calendar application for managing employee tasks, built with **Vue 3** and **TypeScript**.

**Source:** [github.com/SSE-programmer/soft-mall-calendar](https://github.com/SSE-programmer/soft-mall-calendar)

---

## Features

- **Event creation / deletion** with form validation.
- **JSON import / export** — load and save events as JSON files.
- **Date-preloaded event creation** via calendar cell click.
- **Auto-navigation** to event start date upon creation.
- **Real-time current time indicator** with synchronization.
- **View mode selector** — day / week / year views (week view implemented).
- **Keyboard navigation** support via `tabindex`.

## Tech Stack

- **Vue 3** with Composition API.
- **TypeScript** for type safety.
- **Vite** for build tooling.
- **SCSS** for styling.
- No external UI libraries — all calendar components built from scratch.
