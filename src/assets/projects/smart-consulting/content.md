# Smart Consulting — Regional Government Services Portals

**Role:** Software Engineer  
**Type:** Full-time, Hybrid  
**Period:** Jul 2021 — Dec 2023

Development of regional government services portals and social projects deployed across **dozens of Russian regions**.

---

## Responsibilities

- Designed and implemented architectural solutions in collaboration with the lead architect and team lead.
- Independently developed critical platform components.
- Estimated complexity and timelines for user story implementation.
- Decomposed tasks for developers and conducted code reviews.
- Created technical documentation and code reference guides.

## Key Achievements

### Accessibility & UX
- Made all screens **tablet- and mobile-responsive**, increasing mobile user engagement by 30%.
- Implemented accessibility modes for visually impaired users — **3 color schemes, 3 font sizes** via CSS classes, enhancing compliance with **WCAG 2.1** standards.

### Customizable Regional Themes
- Developed **3 switchable UI themes** loaded optionally during frontend bundle builds, enabling flexible branding for different regions and boosting client satisfaction.

### Real-Time MFC Booking
- Built a **live appointment system** with WebSocket-driven slot status updates, reducing failed bookings by 25%.

### Dynamic Form Player
- Redesigned single-service forms into a **multi-step "service story" flow**, allowing users to select pre-configured service bundles instead of manual selections. **Cut form abandonment by 40%** and streamlined document submission.

### Widgets & Integrations
- Created reusable widgets and embedded IFrames for seamless integration of external services, expanding portal functionality with zero-downtime deployments.
